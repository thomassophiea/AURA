Key patterns matched from Connected Clients / Access Points

  • PageHeader — title + subtitle left, toolbar buttons right (Refresh, Columns, Export)
  • Gradient metric cards — 4 for Site Groups, 5 for Sites
  • Search in CardHeader — compound tokenized AND search with result count
  • Table — sticky checkbox column, sortable headers, clickable rows → detail slide-out
  • Sites page filter badge — appears when navigated from Site Groups via site count click, dismissible
  • Pagination — items per page selector + page navigation below table
  • Site count in Site Groups — clickable link that navigates to Sites page with pre-applied filter
