COMPONENT API
<UnifiedFilterBar
  // Search config — always visible
  searchPlaceholder="Search APs by name, serial..."
  searchValue={searchTerm}
  onSearchChange={setSearchTerm}

  // Context selector — defaults tab to page context
  defaultContextTab="access-point"

  // Global filters (all enabled by default)
  showEnvironment={true}
  showTimeRange={true}

  // Page-specific filters (inline after divider)
  extraFilters={<>
    <StatusFilter value={status} onChange={setStatus} />
    <ModelFilter value={model} onChange={setModel} />
  </>}
/>
STATE MANAGEMENT
Global state (useGlobalFilters — persisted, synced)
• site / environment / timeRange / dateFrom / dateTo
• Operational context: mode, siteId, apId, clientId
Local state (owned by each page)
• searchTerm — text input, not persisted
• extraFilters — page-specific dropdowns (band, security, status, model)
Active filter count
• Badge shows count of non-default filters • "✕" resets all to defaults
